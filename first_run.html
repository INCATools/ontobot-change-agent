First Run

Firstly, generate a personalized token as instructed here and save it in src/onto_crawler/token.txt.

Next, run:

$ poetry run ocrawl issues --label synonym

This should return a list of issue titles tagged with the label synonym. By default, we’ve chosen the mondo repository . If you need to test with another repository, add an option -r and follow it with the organization/repository_name.
