Command Line Interface

onto_crawler automatically installs the command onto_crawler. See onto_crawler --help for usage details.

onto_crawler

CLI for onto_crawler.

param verbose

Verbosity while running.

param quiet

Boolean to be quiet or verbose.

onto_crawler [OPTIONS] COMMAND [ARGS]...

Options

-v, --verbose
-q, --quiet <quiet>
--version

Show the version and exit.

issues

onto_crawler issues [OPTIONS]

Options

-r, --repo <repo>

Org/name of the github repo.

-s, --state <state>

State of the issue. [open, close etc.]

-o, --output <output>

Output could be a file or sys.stdout.

-t, --title <title>

Filter based on a search for pattern within title of issue.

-l, --label <label>

Filter based on a search for label of issue.
