Command Line Interface

onto_crawler automatically installs the command ocrawl. See ocrawl --help for usage details.

ocrawl

CLI for onto_crawler.

param verbose

Verbosity while running.

param quiet

Boolean to be quiet or verbose.

ocrawl [OPTIONS] COMMAND [ARGS]...

Options

-v, --verbose
-q, --quiet <quiet>
--version

Show the version and exit.

get-labels

Get all labels for issues.

param repo

GitHub repository name [org/repo_name]

ocrawl get-labels [OPTIONS]

Options

-r, --repo <repo>

Required Org/name of the github repo.

issues

Get issues based on different options.

param repo

GitHub repository name [org/repo_name]

param state

State of issue [“open”, “close” etc.]

param title_search

String lookup in title of issue.

param label

Label lookup for issues.

param output

Output location.

ocrawl issues [OPTIONS]

Options

-r, --repo <repo>

Required Org/name of the github repo.

-s, --state <state>

State of the issue. [open, close etc.]

-o, --output <output>

Output could be a file or sys.stdout.

-t, --title-search <title_search>

Filter based on a search for pattern within title of issue.

-l, --label <label>

Filter based on a search for label of issue.

-n, --number <number>

Filter based on issue number.

process-issue

Run processes based on issue label.

param repo

GitHub repository name [org/repo_name]

param label

Label of issues.

param state

State of issue [“open”, “close” etc.]

ocrawl process-issue [OPTIONS] INPUT

Options

-r, --repo <repo>

Required Org/name of the github repo.

-l, --label <label>

Filter based on a search for label of issue.

-n, --number <number>

Filter based on issue number.

-s, --state <state>

State of the issue. [open, close etc.]

-o, --output <output>

Output could be a file or sys.stdout.

Arguments

INPUT

Required argument
